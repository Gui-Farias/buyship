# Contributing

## Conventional Commits
Formato:
`<type>(<scope>)?: <description>`

Tipos:
- feat: nova funcionalidade (MINOR)
- fix: correção (PATCH)
- docs, chore, test, ci, refactor: sem release por padrão
- feat!: breaking change (MAJOR)

Exemplos:
- feat: adiciona checkout stripe
- fix(cart): corrige remoção de item
- feat!: altera fluxo de checkout
